*Pixel Weather*

Um site de previsão do tempo minimalista com estética Pixel Art, onde a interface se adapta visualmente às condições climáticas 
atuais do usuário. O projeto consome dados em tempo real para fornecer uma previsão de 5 dias com um toque nostálgico de RPGs clássicos.

*Funcionalidades*

- Geolocalização em Tempo Real: Identifica automaticamente a cidade do usuário para fornecer dados precisos.

- Previsão de 5 Dias: Exibição clara de temperatura máxima e condições climáticas.

- Interface Dinâmica: O fundo e os ícones mudam automaticamente de acordo com o clima (Sol, Nublado, Garoa ou Tempestade).

- Design Responsivo: Adaptado para visualização em desktops e dispositivos móveis (Mobile First).

- Estética Retrô: Uso da tipografia "VT323" e artes em pixel para uma experiência de usuário única.

*Tecnologias utilizadas*

- HTML5: Estrutura semântica.

- CSS3: Estilização, Flexbox para layout e Media Queries para responsividade.

- JavaScript: Lógica de consumo de API e manipulação dinâmica do DOM.

- Open-Meteo API: API gratuita para obtenção de dados meteorológicos.

- Google Fonts: Fonte VT323.
